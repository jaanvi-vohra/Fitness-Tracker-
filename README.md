# 💪 ForgeFit

ForgeFit is a modern and responsive **Fitness Tracker Web Application** built using **HTML, CSS, and Vanilla JavaScript**. It helps users monitor workouts, calculate calories burned, track BMI, visualize progress, and stay motivated through personalized fitness insights.

---

## ✨ Features

### 👤 Multi-User Profiles
- Create multiple athlete profiles
- Switch seamlessly between users
- Personalized workout history
- Individual fitness metrics stored separately

---

### 🏋️ Workout Tracking
- Log workouts by exercise type
- Track workout duration
- Automatic calorie calculation
- Delete workout entries
- Persistent workout history

Supported exercises:
- Running
- Cycling
- Weightlifting
- Yoga
- Walking

---

### 🔥 Calorie Calculator

Calories are automatically calculated using:

- Workout duration
- User weight
- MET (Metabolic Equivalent of Task) values

This removes the need for manual calorie estimation.

---

### 📊 Dashboard

Track important fitness statistics including:

- Total Workouts
- Total Calories Burned
- BMI
- Daily Goal Progress

---

### 📈 Workout Analytics

Interactive doughnut chart showing calorie distribution across different workout types using **Chart.js**.

---

### 📏 BMI Calculator

Automatically calculates BMI based on:

- Height
- Weight

Displays BMI category:

- Underweight
- Normal
- Overweight
- Obese

---

### 🎯 Daily Goal Tracking

Set a daily calorie-burning goal.

ForgeFit displays:

- Progress bar
- Percentage completed
- Goal completion status
- Motivational messages

---

### ⏰ Smart Workout Reminder

Interactive analog clock allows users to:

- Set reminder time
- Toggle AM/PM
- Receive browser workout notifications

---

### 🔔 Browser Notifications

Supports workout reminders using the Notification API.

If notifications are blocked, ForgeFit informs the user.

---

### 🌙 Dark Mode

One-click Light/Dark theme toggle.

Theme preference is automatically saved using Local Storage.

---

### 🔍 Filter & Sort

Workout history can be:

- Filtered by workout type
- Sorted by
  - Newest
  - Oldest
  - Highest Calories

---

### 💾 Persistent Storage

ForgeFit stores data using Local Storage:

- User Profiles
- Workout History
- Height
- Weight
- BMI Metrics
- Daily Goals
- Theme Preference

Data remains available after refreshing the page.

---

### 📱 Responsive Design

Designed for:

- Desktop
- Tablet
- Mobile

The layout automatically adapts to different screen sizes.

---

## 🛠️ Technologies Used

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/forgefit.git
```

### Run the Project

Since this project is built using HTML, CSS, and JavaScript only, no installation or dependencies are required.

**Option 1**

Open `index.html` directly in your browser.

**Option 2 (Recommended)**

1. Open the project in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
